# Nmap Port Scan Project

This project demonstrates a basic network reconnaissance task using **Nmap**.

## 🔍 Objective
Scan your local network to identify open ports and understand what services are exposed.

## ⚙️ Tools Used
- Nmap
- (Optional) Wireshark
